ABOUT

経歴
1995年、青森県八戸市生まれ。
中学2年生でギターと出会い、同級生同士でGLAYのコピーバンドを始める。
宮城・仙台の大学へ入学し、在学中にオリジナルロックバンドを結成。
以降「TANAYAN」として活動し、2020年5月にバンド脱退、音楽活動休止。
→バンド公式サイトへ（別タブで開きます）
最近は、テクノロジーが発展した未来の生活についてワクワクしながら過ごしている。

紹介
田中裕也と申します。
　私は、これまでの人生の半分を音楽と共に生きてきました。楽曲制作はもちろん、バンド活動を通してさまざまな経験や制作をしてきました。（主な制作物は「Works」をご覧ください。）
　「誰かに喜んでもらうために何かを生み出し、行動していく」。これには正解が無く、たくさんの試行錯誤、そして気力が必要です。ただ、私にとっては非常にやりがいを感じ、時間を忘れて没頭することも。
　音楽活動を休止した今、私のもともと好きな分野であった「テクノロジー」な業界へ新たに挑戦し、サービス・価値を生み出していきたいと思ってます。また、これまで専門的な知識が無い中での制作が多くありましたが、確かなスキルとして身につけるべく、日々勉強中です。
　ゆくゆくは、「音楽×テクノロジー」として、新たなエンターテイメントの提供を夢見てます。

現職
　携帯キャリアのチャットオペレーターをしています。リアルタイムな文字のみのやり取りの中、お客様の要望や背景、秘めた心情を素早く的確に読み取り、分かりやすく丁寧に伝えることを心がけ、日々応対しています。また、応対をスムーズにするための知識補填も欠かせません。
　ただ、”何かを作る”という自分の得意な分野を活かすことができずにいます。

コメント
バンド活動時代に「Wix」を使って作成したサイト。全35ページほど。
バンドの公式サイトは別でありましたが、バンドの5周年を迎えた記念として作成しました。
サイト作りのノウハウを知らない頃に作成したため、今思うと改良が必要な部分が多くありますが、初めて制作したサイトということもあり、強く思い入れがあります。

主なページの紹介
【「CHANGEOVER」ページ】
5～6周年にバンドがすべきこととして掲げたテーマ「CHANGEOVER」の説明のため。なおこのテーマは、バンド活動のモチベーション維持や、ファンに対して今年どんなことをするのかというワクワク感を与える目的がある。

【全曲視聴ページと各曲の歌詞や説明ページ】
どんな曲があるか、また、どんな思いで曲を制作したかを一覧で見れるようにしたいという思いから作成。また、楽曲アンケートに答えてもらう際に役立ててもらいたいため。

【楽曲アンケートページ】
リリースした曲からファンにとって人気の曲や、今後どのような曲を作ればいいかなどを客観的に知るために作成。また、アンケート結果をTwitterで公開したり、ライブのセットリストの参考にした。

【応援メッセージを送れるページ】
ファンとの交流のため。送られてきたメッセージは全文管理しメンバーへ共有。

【5周年ライブ特設ページ】
5周年の節目として行ったライブイベントを盛り上げるため、ライブ実施1ヶ月前に作成し公開。各出演者の紹介、ライブの詳細情報、新グッズ、ライブ中の写真など。

SKILL

HTML5/CSS3

主に「Progate」と「ドットインストール」を利用し約３ヶ月が経ちました。1から作れと言われると、まだまだ時間がかりますが、コードの意味はある程度理解してます。CSSでデザインが変化する感じが楽しいです。

JavaScript

ProgateにあるJavaScript教材はすべて学びました。また、Node/jsやSQLについても少々学びました。実践的な事はまだまだですが、CSSではできない動きを加えることができ、今後より一層学んでいきたい言語です。

Java

開発における大半が「Java」でできているとの情報から、Progateを利用して、一通り学びました。Javaを学ぶ前にJavaScriptを学んでいたためか、早く理解することができました。

Photoshop / Illustrator / Lightroom

バンド活動時代に簡単な画像制作の際に利用し、基本的な操作は可能です。デザイン作成などのため、今後しっかりと扱い方を習得していきたいと思います。なお、写真加工に特化した「Lightroom」では100枚以上加工した経験があり、写真加工のノウハウはあります。

Premiere Pro

バンド活動期に利用しており、これまで10動画ほど作成しました。カットやテロップなど基本的な操作であれば問題なく可能です。他、「After Effects」を用いて、簡単なアニメーションを作成したことがあります。
音に関して自信があるため、効果音でのイコライジングやリバーブ等による加工も問題なく可能です。

Excel

現職で日頃から利用してます。仕事の「効率化」を目指し、さまざまな関数や条件付き書式を用いた資料などを多数作成しました。
主に、プルダウンからプランを選択するのみで料金のシミュレーションができる資料。また、年月を特定のセルに入力するのみで、所属部署全体の1ヶ月分のシフト表ができるExcelを作成をしました。今では社内全体でそれらを利用してます。

Cubase

音楽制作ソフトとして、約8年ほど利用し、これまで作成した曲は100を超えると思います。基本的な操作はもちろん、ミックスやマスタリングも可能。作詞以外の工程全てを担当した曲をYouTubeで公開した経験もあります。
WEB制作を勉強していった後、音と融合したエンターテイメントチックなサイトをいつか制作してみたいです。
